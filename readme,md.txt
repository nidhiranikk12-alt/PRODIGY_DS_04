# PRODIGY_DS_04 – Sentiment Analysis of Social Media Data

## Task Overview

This project was completed as part of my Data Science Internship at Prodigy InfoTech.

The objective of this task is to analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes towards different topics and brands.

## Dataset

**Dataset:** Twitter Training Dataset

The dataset contains:

* Tweet ID
* Topic/Brand
* Sentiment Label
* Tweet Text

### Sentiment Categories

* Positive
* Negative
* Neutral
* Irrelevant

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* WordCloud

## Project Workflow

1. Load and explore the dataset.
2. Clean and preprocess the data.
3. Analyze sentiment distribution.
4. Visualize sentiment patterns using charts.
5. Generate Word Clouds for Positive and Negative sentiments.
6. Perform topic-wise sentiment analysis.
7. Extract insights from social media discussions.

## Visualizations Created

### 1. Sentiment Distribution

A count plot showing the distribution of sentiment categories across the dataset.

### 2. Positive Sentiment Word Cloud

A visualization of the most frequently used words in positive tweets.

### 3. Negative Sentiment Word Cloud

A visualization of the most frequently used words in negative tweets.

### 4. Topic-wise Sentiment Analysis

A comparative visualization showing sentiment distribution across the most discussed topics and brands.

## Output Files

